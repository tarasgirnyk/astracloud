Меню кабінету — фінальна структура

Єдиний механізм навігації: у хедері — Головна (маркетинговий сайт), VPS сервери, Виділений сервер, Колокація — поточна сторінка підсвічена. Кошик, мова та баланс завжди на видноті поза випадним списком. «Ласкаво просимо ▾» розкриває весь кабінет одним списком із чіткими заголовками груп і роздільниками. На мобільному той самий список зібраний у розгортні секції по групах. Іконки — лінійний нейтральний набір замість emoji.

Звузьте вікно нижче 1100px, щоб побачити мобільну панель (гамбургер зліва вгорі).

Що змінилось у цій версії (з Claude Design)

  • Мобільна панель — розбита на розгортні секції (Управління / Обліковий запис / Підтримка) замість одного довгого списку. «Управління» відкрита за замовчуванням.
  • Десктопний випадний список — без accordion, лише чіткіші заголовки груп і роздільники між ними.
  • Іконки — замінено emoji на нейтральний лінійний SVG-набір (24×24, stroke 1.75), реалізовано без залежності від рантайму Claude Design — чистий CSS/HTML (hover для десктопу, checkbox-акордеон для мобільного, без JS).
  • Активна сторінка — приклад підсвітки поточного пункту в хедері й мобільній панелі («VPS сервери»).
  • Вихід — внизу списку, кольором error, без діалогу підтвердження.
  • Брейкпоінт — мобільна панель тепер вмикається на ≤1100px (було 768px) — з кошиком/мовою/балансом у хедері на це більше місця не вистачає раніше.
