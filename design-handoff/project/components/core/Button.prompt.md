Pill-shaped call-to-action button; use `primary` for the single site-wide CTA color (orange gradient), `secondary`/`ghost` for lower-emphasis actions on light sections, `ghost-dark`/`dark` on navy sections.

```jsx
<Button variant="primary" size="lg">Спробувати</Button>
<Button variant="secondary">Наші послуги</Button>
<Button variant="ghost-dark">Дізнатися більше</Button>
```

Variants: `primary`, `secondary`, `ghost`, `ghost-dark`, `dark`. Sizes: `sm`, `md`, `lg`. Pass `icon` for a leading icon element, `disabled` to disable.
