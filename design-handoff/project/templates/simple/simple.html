Політика конфіденційності

Astra Cloud поважає право користувачів на конфіденційність та забезпечує захист персональних даних відповідно до чинного законодавства України.

Користуючись сервісами Astra Cloud, ви погоджуєтесь з умовами цієї політики.

Які дані ми збираємо

Ми можемо збирати наступні дані:

  • ім'я та контактні дані (email, телефон)
  • платіжну інформацію (через платіжні системи)
  • IP-адресу та технічні дані підключення
  • дані облікового запису та історію використання сервісів

Як ми використовуємо дані

Зібрані дані використовуються для:

  • надання та підтримки сервісів
  • обробки платежів
  • зв'язку з клієнтом
  • забезпечення безпеки та запобігання зловживанням
  • покращення якості послуг

Захист даних

Astra Cloud застосовує технічні та організаційні заходи для захисту інформації від несанкціонованого доступу, втрати або зміни.

Передача даних третім особам

Ми можемо передавати дані третім сторонам виключно у таких випадках:

  • для обробки платежів
  • на вимогу законодавства України
  • для забезпечення роботи сервісів

Ми не продаємо персональні дані третім особам.

Зберігання даних

Дані зберігаються протягом строку, необхідного для надання послуг та виконання зобов'язань.

Права користувача

Користувач має право:

  • отримати інформацію про свої дані
  • вимагати їх зміну або видалення
  • обмежити обробку даних у передбачених законом випадках

Контакти

З питань конфіденційності ви можете звернутися за адресою: support@astra.in.ua

Astra Cloud залишає за собою право змінювати цю політику з попереднім повідомленням.
