If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.
If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Key things to know before you start:
- Astra Cloud is a friendly-but-reliable Ukrainian cloud hosting brand (VPS, dedicated servers, colocation) with an astronaut mascot — no mascot image asset exists yet, never draw or generate one; use a labeled placeholder and ask the client for the real illustration.
- Orange is the only CTA/accent color; navy gradient dark sections alternate with warm off-white (`#faf9f5`) light sections; cyan/violet are decoration-only (glow, gradients), never text.
- Unbounded for headlines, Inter for everything else.
- Copy is formal "ви", light space-themed puns dosed sparingly, no emoji in real copy.
